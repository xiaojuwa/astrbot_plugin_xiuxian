一款为 AstrBot 设计的放置类文字修仙游戏插件。从凡人踏入仙途，历经修炼、战斗、探索，终成大道！

> 感谢 [@Zhalslar](https://github.com/Zhalslar) 提供的模块化思路重构了整个插件的结构

---

## 特色功能

| 特色 | 描述 |
|------|------|
| **完整成长体验** | 28级境界体系，从"炼气一层"到"祭道之上" |
| **动态战斗系统** | 基于标签系统动态生成怪物和Boss，每次战斗独一无二 |
| **丰富物品系统** | 60+ 种物品，涵盖丹药、材料、法器、功法 |
| **炼丹/炼器系统** | 收集材料炼制丹药和法器，10级炼丹师/炼器师等级 |
| **宗门社交** | 创建或加入宗门，与道友共谋发展 |
| **多元化玩法** | 排行榜、每日任务、奇遇探索、天劫挑战、悬赏任务 |
| **高度可配置** | 所有数值、物品、怪物、配方均可通过JSON文件自定义 |
| **专业数据管理** | aiosqlite独立存储 + 版本化数据库迁移系统 |

## 快速开始

### 安装

1. 在 AstrBot 插件市场搜索 `astrbot_plugin_xiuxian` 安装
2. 或手动克隆到 `AstrBot/data/plugins/` 目录

### 开启修仙之路

```
发送: 我要修仙    → 创建角色
发送: 修仙帮助    → 查看所有指令
发送: 签到        → 领取每日奖励
```

### 境界体系

```
炼气期(5层) → 筑基期(3期) → 搬血境 → 洞天境 → 化灵境 → 铭纹境 →
列阵境 → 尊者境 → 神火境 → 真一境 → 圣祭境 → 天神境 → 虚道境 →
斩我境 → 遁一境 → 至尊境 → 真仙境 → 仙王境 → 准仙帝 → 仙帝境 → 祭道境 → 祭道之上
```

### 丹药中毒系统 (v2.5.3新增)

服用丹药时需注意用量，过量服用会导致丹毒入体！

| 服用数量 | 中毒概率 | 说明 |
|----------|----------|------|
| 1-3 颗 | 0% | 安全范围 |
| 4 颗 | 10% | 开始有风险 |
| 5 颗 | 20% | |
| 6 颗 | 30% | |
| 7 颗 | 40% | |
| 8 颗 | 50% | |
| 9 颗 | 60% | |
| 10+ 颗 | 100% | 必定中毒 |

**中毒效果：**
- 丹药效果完全失效（不获得任何增益）
- 当前血量扣除 50%（至少保留1点血）
- 丹药仍然被消耗

**不受影响的丹药：**
- 逆天改命丹（特殊丹药，只能单颗使用）

## 配置说明

### WebUI 配置

通过 AstrBot WebUI 的插件配置页面可调整：

| 配置项 | 描述 |
|--------|------|
| `ACCESS_CONTROL.WHITELIST_GROUPS` | 白名单群号列表，留空表示所有群可用 |
| `VALUES.INITIAL_GOLD` | 新玩家初始灵石数量 |
| `VALUES.CHECK_IN_REWARD_MIN/MAX` | 签到奖励范围 |
| `VALUES.BASE_EXP_PER_MINUTE` | 闭关每分钟获得的基础修为 |
| `VALUES.CREATE_SECT_COST` | 创建宗门消耗灵石 |
| `VALUES.SHOP_DAILY_ITEM_COUNT` | 每日商店刷新商品数量 |
| `SPIRIT_ROOT_SPEEDS.*` | 各灵根修炼速度倍率 |

### 配置文件

| 文件 | 用途 |
|------|------|
| `config/level_config.json` | 境界配置（名称、升级修为、成功率） |
| `config/items.json` | 物品配置（名称、价格、效果） |
| `config/monsters.json` | 怪物模板配置 |
| `config/bosses.json` | Boss模板配置 |
| `config/tags.json` | 怪物标签系统（属性修饰、掉落） |

### v2.6.1

**秘境系统重新设计 - 重大更新**

**新增功能：**
- **8种事件类型**：不再只是战斗和宝箱
  - 普通怪物：基础战斗
  - 精英怪物：强化版怪物(属性+20-30%)，奖励1.5倍
  - Boss：最终层强大Boss
  - 宝箱：获得灵石奖励
  - 陷阱：可能造成伤害和灵石损失
  - 祝福/诅咒：随机获得正面或负面buff效果
  - 选择事件：玩家需做出策略选择（分岔路口、可疑宝箱等）
  - 商人事件：购买道具或服务
  - 神秘事件：随机好坏事件

- **5种主题秘境**：每种主题有不同的事件分布
  - 试炼之地：平衡型，适合稳定探索
  - 宝藏密室：宝箱丰富(40%)，但陷阱众多(20%)
  - 妖兽巢穴：战斗密集(50%)，精英怪多(15%)
  - 古老遗迹：神秘事件多(10%)，可能获得珍稀奖励
  - 幽冥鬼域：极度危险，奖励丰厚，诅咒概率高(30%)

- **3个难度等级**：风险与收益的平衡
  - 普通：消耗1.0x，奖励1.0x
  - 困难：消耗1.5x，奖励2.0x
  - 地狱：消耗2.0x，奖励3.0x

- **策略选择系统**：
  - 分岔路口：选择高风险高回报、平衡或安全路线
  - 可疑宝箱：直接开(可能陷阱或大奖)、小心检查或放弃
  - 商人交易：购买疗伤丹药、buff丹药或随机物品

**玩法改进：**
- 秘境不再是无脑刷怪，需要策略决策和资源管理
- 楼层难度动态调整：前30%楼层降低陷阱和精英概率
- 完成奖励：根据难度给予额外灵石奖励
- 事件描述更生动，增强沉浸感

**新增指令：**
- `探索秘境 [类型] [难度]` - 指定秘境类型和难度
  - 示例：`探索秘境` (默认试炼普通)  
  - 示例：`探索秘境 宝藏 困难` (宝藏密室困难难度)
- `选择 <数字>` - 在选择事件或商人事件中做出选择

**技术改进：**
- 新增 `realm_events.py` 模块：事件生成和处理逻辑
- 重构 `RealmGenerator`：支持主题和难度
- 扩展数据模型：支持事件选择和待选择状态
- 优化奖励计算：应用难度倍率和主题修正

### v2.5.3

**新增机制：**
- **丹药中毒系统**：防止玩家一次性服用大量丹药
  - 单次服用 1-3 颗丹药：安全，无中毒风险
  - 单次服用 4-9 颗丹药：中毒概率递增（4颗10%，5颗20%...9颗60%）
  - 单次服用 10 颗及以上：必定中毒（100%）
  - 中毒效果：丹药效果完全失效 + 当前血量扣除50%
  - 适用于所有丹药类型（buff丹药、回血丹、修为丹等）
  - 特殊丹药（如逆天改命丹）不受影响
